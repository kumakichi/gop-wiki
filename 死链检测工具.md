# 下载

* Linux 64bits: http://open.qiniudn.com/badlinks-linux-amd64.tar.gz
* Mac OS: http://open.qiniudn.com/badlinks-darwin-amd64.tar.gz

# 使用方法

```bash
badlinks input-file1 [... input-fileN]
```

样例：

```
badlinks https://support.qiniu.com/hc/kb/article/68873/
```
